# Explainable AI (XAI) for Emotion Recognition

## Overview
This repository contains an in-depth analysis of Explainable AI (XAI) methodologies applied to Natural Language Processing (NLP). The core notebook (`XAI.ipynb`) investigates the behavior, feature importance, and decision-making processes of machine learning models before and after integrating interpretability frameworks. 

The primary focus is on enhancing transparency for emotion recognition systems, specifically analyzing how models interpret complex linguistic data.

## Key Features
* **Model Baseline & Comparison:** Evaluates initial model predictions and performance metrics.
* **XAI Integration:** Implements explainability frameworks (e.g., SHAP, LIME) to demystify "black-box" model outputs.
* **Feature Importance Analysis:** Visualizes the specific words, tokens, or features that drive the model's emotional classifications.
* **Transparent Affective Computing:** Bridges the gap between high-accuracy NLP models and human-interpretable results.
